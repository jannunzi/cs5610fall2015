Block & Inline Elements

Block Elements

Block elements break up the content into vertical rectangles that occupy the entire width of the parent container and the height of its content and start a new line.

Example block elements are

  • H* - Head elements
  • P - paragraph elements
  • DIV - generic block element

Inline Elements

Inline elements are rendered inline with other surrounding elements. Inline elements do not create a new line and its width is only as wide as its content. Example inline elements are:

  • B - Bold
  • A - Anchor
  • IMG - Image
  • SPAN - Generic inline element
