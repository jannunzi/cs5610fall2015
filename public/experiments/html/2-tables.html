TABLE element is used to format and display tabular data. Never use a TABLE element to layout a page.

Table Anatomy

Tables are defined with the TABLE element. Within the TABLE element you can include the following 3 elementS:

  • THEAD
  • TBODY
  • TFOOTER

Table Rows

Use the TR to declare table rows. Use the TD to declare table columns. Use the TH to declare table headings.

Sample Table

Here's a simple table

Courses

Course Title Available Seats Start Date
Java 101 25 10/9/2015
C# 25 10/9/2015
Python 25 10/9/2015 
<table>
    <thead>
        <tr>
            <th>Course Title</th>
            <th>Available Seats</th>
            <th>Start Date</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Java 101</td>
            <td>25</td>
            <td>10/9/2015</td>
        </tr>
        <tr>
            <td>C#</td>
            <td>25</td>
            <td>10/9/2015</td>
        </tr>
        <tr>
            <td>Python</td>
            <td>25</td>
            <td>10/9/2015</td>
        </tr>
    </tbody>
</table>

Table Borders

Use border attribute to set table's border 

<table border="1">
Course Title Available Seats Start Date
Java 101 25 10/9/2015
C# 25 10/9/2015
Python 25 10/9/2015

Table Width

Use width attribute to set the table's width to pixels or percentage of window. 

<table width="100%" border="1">
Course Title Available Seats Start Date
Java 101 25 10/9/2015
C# 25 10/9/2015
Python 25 10/9/2015

Row and Column Span

<th rowspan="2">Semester</th>
<th colspan="2">Course</th>
Semester Course
ID Title
Spring 2015 CS5610 Web Development
CS5200 Database Management Systems

Cellpadding & Cellspacing

Configure the space between cells and padding within cells with cellpadding and cellspacing attributes. 

<table border="1" width="100%" cellspacing="10px" cellpadding="10px">
Semester Course
ID Title
Spring 2015 CS5610 Web Development
CS5200 Database Management Systems
